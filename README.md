# CreamDream

Backend pentru un storefront al unei gogosarii.

Functii:
- Posibilitate de a comanda ca si utilizator autentificat
- Gestionare interna drept staff/admin


Structura:

```
.
│
└───CreamDream
    ├───Controllers
    ├───Database
    │   └───Models
    ├───DTO
    │   ├───Create
    │   ├───Entity
    │   └───Update
    └───Services
        ├───Abstractions
        └───Implementations
```
